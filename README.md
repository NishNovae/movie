# Movie

## Installation
```bash
# main
$ pip install git+https://github.com/NishNovae/movie.git

# branch
$ pip install git+https://github.com/NishNovae/movie.git@<BRANCH>
```

## Setting up dev environment
```bash
$ git clone <url>
$ cd <DIR>

# option
$ pdm venv create
$ source .venv/bin/activate
```
